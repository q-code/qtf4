Les <b>tags</b> sont des marqueurs personnels que l'on peut attacher aux sujets. Ces tags sont utilisés pour grouper les sujets ou générer des statistiques particulières. Les tags que l'utilisateur peut ajouter aux sujets sont libres. Cependant, si une liste de tags est définie, celle-ci s'affichera dans un menu déroulant. Voyez la <a href="qtf_adm_tags_help.php" target="_blank">page d'aide</a> pour en savoir plus.
