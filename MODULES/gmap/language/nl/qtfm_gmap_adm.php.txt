U hebt een Google map API sleutel nodig om de kaartfuncties te gebruiken. Om deze sleutel te krijgen, moet u voor het gebruiken van de kaart API van Google op uw website omhoog ondertekenen (zie  <a href="https://console.cloud.google.com">Google Console</a>). Laat API sleutel leeg om de kaartfuncties onbruikbaar te maken. Kaartservices zijn aanvullende API's die in uw Google-console moeten worden geactiveerd.
